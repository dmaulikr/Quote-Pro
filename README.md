# **Quote Pro**
### Quote Pro is a Random Quote Maker using 2 APIs.
- Quotes are generated using https://forismatic.com/en/api/.
- Images are generated using http://lorempixel.com/.

# Preview

### Saved Quotes
- Quotes are saved/persisted using Realm.
